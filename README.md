# 秒记账 v2.3 CSS启动页版

本版修改：
- 启动页不使用 PNG，完全由 HTML/CSS 绘制
- 去掉原生覆盖启动页，避免挡住页面导致无法点击
- 启动页 1.6 秒后自动进入首页
- 保留预算首页、记账、记录修改/删除、分类、统计、我的、备份导出
- 返回键可关闭弹窗/面板/回首页

上传 GitHub 后运行 Actions，下载 `miaojizhang-v2-2-apk`。

修复：启动页跳转独立于业务 JS，避免卡在启动页。


## v3.3.0 更新
- 启动页改为手写账单风格，完整人名以低存在感水印/纸面标记融入。
- 数据备份与恢复统一为 JSON 格式，移除 CSV 导出入口。
- 首页日历改为 年 → 月 → 日：先看 12 个月消费，点月份后看每日消费。
- 头像点击改为从相册选择照片，并压缩保存到本地。
- 记一笔默认日期改为今天。
